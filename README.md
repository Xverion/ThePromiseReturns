# Project Second Promise

A Dalamud plugin for Final Fantasy XIV that spawns Koana as a persistent client-side NPC companion.

## Features

- Spawns Koana as an NPC that only you can see
- Auto-spawns in Tuliyollal
- Interactive dialogue using the native Talk window
- Click-to-advance dialogue progression

## Installation

1. Open Dalamud Settings with `/xlsettings`
2. Go to the **Experimental** tab
3. Under **Custom Plugin Repositories**, add:
   ```
   https://raw.githubusercontent.com/Xverion/ThePromiseReturns/main/repo.json
   ```
4. Click the **+** button, then **Save and Close**
5. Open the Plugin Installer with `/xlplugins`
6. Search for **Project Second Promise** and click **Install**

## Commands

| Command | Description |
|---------|-------------|
| `/koana` | Opens the plugin window |
| `/spawnkoana` | Manually spawn Koana |
| `/despawnkoana` | Dismiss Koana |
